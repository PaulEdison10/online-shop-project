# 🛍️ PHP-MySQL Admin Dashboard – PictusCode

A fully functional **Admin Dashboard** built with PHP, MySQL, and Bootstrap to manage shops and products. Features animated backgrounds, glassmorphism UI, and dynamic data handling.

## 📸 Preview

![Dashboard Preview](assets/images/screenshot.jpg)

---

## ✨ Features

✅ Admin login/logout system  
✅ Add / Edit / Delete **Shops**  
✅ Add / Edit / Delete **Products** with image upload  
✅ View all shops and their respective products  
✅ Stylish **glassmorphism cards**  
✅ Animated background using CSS  
✅ Responsive design with **Bootstrap 5**

---

## 🛠️ Tech Stack

- ✅ PHP (Backend)
- ✅ MySQL (Database)
- ✅ Bootstrap 5 (Frontend)
- ✅ HTML5 & CSS3 (Structure & Styling)
- ✅ JavaScript (Effects & Interactions)
- ✅ Optional: particles.js or animated backgrounds
